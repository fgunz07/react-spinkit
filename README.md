# React Spinkit
A simple react plugin create from [spinkit](https://github.com/tobiasahlin/SpinKit)

#### Installation
```cmd
	npm i react-spinkit
	yarn add react-spinkit
```

#### Usage
```javascript
	// import component
	import { Flipbox } from 'react-spinkit'

	// pass optional props

	<Flipbox 
	height="40px"
	width="40px"
	margin="100px auto"/>
```

A very light loading icons.
